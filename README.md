# Gekke Mieren

Gekke Mieren is a generative art project that simulates ants using the Slime Mould algorithm. The project allows for some user input to influence the behavior and patterns generated by the simulated ants. The name "Gekke Mieren" (Crazy Ants) is inspired by the documentary "De Gekke Mieren van Nyma" which explores the fascinating behavior of ants.

## Generative Art Engine
The core of the project is a generative art engine that simulates ant behavior using the Slime Mould algorithm. This algorithm mimics the way slime moulds grow and optimize their networks, creating organic and intriguing patterns.

## User Interaction
While the project primarily generates art autonomously, users can provide input to influence certain parameters of the simulation, allowing for a degree of creative control over the generated patterns.

## Simulation
The simulation component handles the behavior of the ants, their movement, and interactions. It uses the Slime Mould algorithm to create evolving patterns that can be visualized as art.

## Visualization
The project includes visualization tools to render the patterns generated by the ant simulation. These visualizations can be saved or displayed in real-time.

## Tools

* `cargo run` compiles and runs the project locally, allowing you to see the generative art in action.

## Quickstart

Requirements:

* Rust and Cargo installed on your system.
* Basic knowledge of running Rust projects.

### Install

In order to install the dependencies and prepare the project, run

    cargo build

This installs the necessary dependencies and compiles the project.

### Run

After installing the dependencies, run the project with

    cargo run

This starts the generative art simulation locally.

### Test

To run tests (if available), use

    cargo test

### Release

To prepare a release build, run

    cargo build --release

This optimizes the build for performance.
